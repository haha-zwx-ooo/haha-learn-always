| 数据结构    | 占用内存          | 默认值   |
|---------|---------------|-------|
| byte    | 1字节8位         | 0     |
| short   | 2字节16位        | 0     |
| int     | 4字节32位        | 0     |
| long    | 8字节64位        | 0     |
| float   | 4字节32位        | 0     |
| double  | 8字节64位        | 0     |
| char    | 2字节16位        | u0000 |
| boolean | 1位有的是4字节int标识 | false |
